  prepended to all Markdown relative \texttt{url}s. It must be a valid
  \texttt{url} without query or fragment parts (that is,
  \emph{http://lisp.org/doc/} but not \emph{http://lisp.org/doc?a=1} or
  \emph{http://lisp.org/doc\#fragment}). Note that intra-page links
  using only \texttt{url} fragments (e.g. and explicit HTML links (e.g.
  \texttt{\textless{}a\ href="..."\textgreater{}}) in Markdown are not
